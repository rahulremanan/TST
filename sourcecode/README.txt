00_Install_Libraries.R contains R code for instllation of dependent libraries from an online CRAN mirror.
00_Install_Libraries_Offline.R contains R code for offline installation of dependent packages inside ~\library folder.
01_Stats.R file contains R code for statistical tests on the TST.csv dataset.
02_ML.R file contains R code for running a random forest ensemble learning prediction on TST.csv dataset.
All outputs are written to the default working directory.
Use: getwd() to identify the working directory for R.

NOTE: Outputs generated using the RdRAND library simultaneosuly with the Marsenne Twister pseudorandom generator is available upon request from Rahul Remanan (rahul@remanan.net).
